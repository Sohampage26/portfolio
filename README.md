# Soham Page — Portfolio

A personal portfolio website built with vanilla HTML, CSS, and JavaScript. Features a dark/light mode toggle, particle network background, custom cursor, scroll reveal animations, and a fully functional contact form powered by Formspree.

## Live Site

[sohampage26.github.io/portfolio](https://sohampage26.github.io/portfolio)

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Hero section with typewriter animation, stats bar |
| `about.html` | About me, work experience timeline |
| `skills.html` | Skill bars, tech stack pills, language cards |
| `projects.html` | Projects with filter, freelance work, achievements |
| `certifications.html` | Certifications and education timeline |
| `contact.html` | Contact info and live Formspree-powered form |

## Tech Stack

- **Frontend** — HTML5, CSS3, Vanilla JS
- **Fonts** — Cabinet Grotesk, Plus Jakarta Sans (Google Fonts)
- **Animations** — CSS keyframes, IntersectionObserver scroll reveal
- **Background** — Canvas particle network
- **Contact form** — Formspree
- **Hosting** — GitHub Pages

## Features

- Dark / light mode with `localStorage` persistence
- Animated particle network background
- Custom cursor with hover scaling effect
- Scroll reveal animations with staggered grid entries
- Project filter by category (ML, Web, Data, DSA, Freelance)
- Project detail modals with highlights
- Freelance section — 2 live clinic digitisation projects
- JLPT N4 Japanese language section
- Fully functional contact form (emails delivered via Formspree)
- Responsive layout for mobile and desktop
